# AI Career Recommendation System (Java)

This project is a Java-based AI Career Recommendation System that suggests suitable career paths based on user skills and interests.

## Features
- Skill-based career recommendations
- Rule-based AI decision logic
- Interactive console application
- Clean and extendable Java code

## Technologies Used
- Java
- Basic AI decision logic

## How It Works
1. User inputs skills and interests
2. System analyzes input using predefined rules
3. A suitable career path is recommended

## How to Run
```bash
javac CareerRecommendation.java
java CareerRecommendation
